# Refactorización HTML5 - Clínica Dental Sonrisa

## Descripción
Examen de refactorización de una página web de clínica dental usando etiquetas semánticas de HTML5.

## Objetivos
- Mejorar la semántica del código.
- Mantener el diseño original.
- Mejorar accesibilidad y mantenimiento.
- Aplicar buenas prácticas HTML5.

## Tecnologías utilizadas
- HTML5
- CSS3

## Mejoras realizadas
- Sustitución de `div` por etiquetas semánticas.
- Mejora de la estructura de navegación.
- Organización más clara del contenido.
- Código más limpio y escalable.
